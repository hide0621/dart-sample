# dartのnoSuchMethod()について

・noSuchMethodメソッドは、未定義のメソッドが呼び出された際に実行される

・ただし呼び出し元のオブジェクト（レシーバー）がdynamic型か、そのオブジェクトが未実装のメソッドを持っていて、かつnoSuchMethodメソッドがオーバーライドされていて、ObjectクラスのnoSuchMethod と実装が異なっている場合に限る
